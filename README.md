# 🎵 XP Music Player

Tocador de músicas no estilo do Windows Media Player do Windows XP.
Cole um link do YouTube e ouça direto no navegador.

## Tecnologias
- HTML
- CSS
- JavaScript

## Como usar
1. Acesse o link do GitHub Pages
2. Cole um link do YouTube no campo
3. Clique em Load e aproveite!

## Aviso
Como o player usa links do YouTube, ele pode bloquear reprodução em domínios não autorizados.
