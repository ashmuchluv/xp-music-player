# 📻 XP Music Player

> "A nostalgia do Windows Media Player clássico, reconstruída para a web moderna."

⚠️ **Nota:** Este projeto está atualmente **em desenvolvimento**. Novas funcionalidades, ajustes visuais e refinamentos de código estão sendo implementados continuamente.

Este projeto é uma reconstrução da interface clássica do player de música do Windows XP. Desenvolvido com HTML, CSS e JavaScript puro, o foco principal é unir a engenharia reversa de design com a manipulação prática de áudio e mídia na web.

---

## 🚀 Funcionalidades (Atuais e Planejadas)

* **Controles de Mídia:** Gerenciamento de estados essenciais como `Play`, `Pause`, `Stop` e controle linear de volume.
* **Inserção de Links:** Campo dinâmico preparado para carregar e reproduzir fluxos de áudio externos diretamente na aplicação.
* **Visualizador Gráfico:** Área dedicada para renderização de animações visuais (espectro de áudio) utilizando o elemento `canvas` do HTML5.
* **Atalhos de Teclado:** Mapeamento de teclas nativas (como a barra de espaço e setas) para controlar o player sem o uso do mouse.

---

## 🎨 Diferenciais de UI/UX

* **Fidelidade Visual:** Uso da tipografia clássica `Tahoma` e posicionamento preciso para sobrepor os elementos exatamente no layout do software original.
* **Escala Dinâmica:** Lógica via JavaScript que calcula o tamanho da tela e ajusta o tamanho do player automaticamente, impedindo que a interface quebre em diferentes resoluções.
